# 🎬 MediaToolkitPro

**MediaToolkitPro** is a modern, fast, and completely free all-in-one utility for Windows, designed to handle your daily media files with a beautiful and intuitive user interface built on **WinUI 3**.

No more uploading your private files to shady online converters. Everything is processed locally on your machine at lightning speed! ⚡

## ✨ Features

* 🗜️ **Compress:** Drastically reduce the size of your Videos and Photos with one click (Medium/High presets).
* 🔄 **Convert:** Seamlessly convert between video, audio, image formats, and even documents (Word ↔️ PDF).
* ✂️ **Trim:** Cut your long videos or audio files accurately with a built-in live preview player.
* 📐 **Crop:** Crop your videos or images visually using an intuitive drag-and-draw canvas.

## 🚀 How to Download & Install

You don't need to be a programmer to use this! 
1. Go to the **[Releases](#)** section on the right side of this page.
2. Download the latest `MediaToolkitPro_v1.0.zip` file.
3. Extract the folder anywhere on your computer (e.g., your Desktop).
4. Double-click on `MediaToolkitPro.exe` to launch the app!

*(Note: On the first launch, Windows SmartScreen might show a blue warning because this is a new app. Click "More info" -> "Run anyway". Also, the app will automatically download the required FFmpeg engine in the background during your first video operation).*

## 🛠️ Built With
* **C# / .NET 8**
* **WinUI 3 (Windows App SDK)** for the modern Windows 11 Fluent Design.
* **Xabe.FFmpeg** for blazing-fast media manipulation.
* **FreeSpire.Doc / FreeSpire.PDF** for document conversions.
